# Shanmukh's Portfolio

This is an interactive portfolio built with React, Vite, and Three.js.

## Setup

1.  Install dependencies:
    ```bash
    npm install
    ```

2.  Run the development server:
    ```bash
    npm run dev
    ```

3.  Open your browser at `http://localhost:5173`.

## Features

-   **Interactive Background**: 3D Force Directed Graph representing skills and connections.
-   **Dark/Light Mode**: Toggle between themes.
-   **Responsive Design**: Works on mobile and desktop.
-   **Animations**: Smooth scroll and entry animations using Framer Motion.
